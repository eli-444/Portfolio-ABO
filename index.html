Machine Learning


Définition

L'apprentissage automatique (machine learning) est une branche de l'intelligence artificielle qui repose sur des méthodes mathématiques et statistiques. Son objectif est de permettre aux ordinateurs d'« apprendre » à partir de données, améliorant ainsi leurs performances sans être explicitement programmés pour chaque tâche.


Comment fonctionne le Machine Learning ?

Le machine learning fonctionne en collectant d'abord des données brutes (images, sons, textes...). Ensuite, ces données sont classées pour obtenir des résultats pertinents. Des outils comme CoreML (Apple) et TensorFlow sont utilisés pour entraîner les modèles d'apprentissage, qui peuvent être intégrés dans des applications et améliorés au fil de leur utilisation.


À quoi sert le Machine Learning ?

Le machine learning permet notamment la reconnaissance d’images, la reconnaissance vocale (Shazam, Siri, Google Assistant), ainsi que l'analyse et la classification automatique de données.


Applications courantes

  • Vision par ordinateur (reconnaissance faciale, analyse d'images et vidéos)
  • Reconnaissance de caractères manuscrits (OCR, codes postaux...)
  • Détection de fraudes bancaires
  • Prédiction des marchés financiers
  • Jeu vidéo et intelligence artificielle (AlphaGo, Deep Blue...)
  • Robotique et assistance médicale
