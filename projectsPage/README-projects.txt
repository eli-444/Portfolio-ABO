Projects Facade — Aurora Variants
=================================

Fichiers ajoutés :
- dynamic-projects.css : Thème dynamique + aurora animé.
- project.js           : Interactions (toggle thème, reveal, parallax).

Chaque page reçoit une palette aurora différente via l’attribut data-aurora sur <body> :
  ocean  (bleu)
  emerald (vert)
  royal  (violet)
  crimson (rouge)
  sunset  (orange/rose)

Mapping automatique appliqué :
- gokuvscell.html -> emerald
- gsb.html -> sunset
- kira.html -> crimson
- osher.html -> ocean
- tensen.html -> royal
